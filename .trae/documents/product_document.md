## 1. 产品概述

Smart Timeline（智能多轨时间轴）是一个独立的多轨甘特图/时间轴可视化 Web 应用，从 Obsidian 插件重构而来。它提供全年垂直堆叠日历视图，支持任务、分组、便签、里程碑四种数据实体的完整 CRUD 管理，并通过智能行合并算法自动避免任务重叠，跨月任务自动切割渲染，保持视觉连续性。

- **目标用户**：需要可视化时间规划的个人用户（如考研复习规划、项目管理等）
- **核心价值**：以优雅的莫兰迪色系 UI 呈现全年任务时间线，零学习成本即可上手
- **数据持久化**：LocalStorage 实时保存 + JSON 文件导入/导出

## 2. 核心功能

### 2.1 功能模块

1. **时间轴视图页**：全年12月份垂直堆叠日历，任务条渲染，今日标记，周末高亮
2. **任务管理**：新建/编辑/删除任务，支持主线任务标识、完成标记、自定义颜色
3. **分组管理**：任务分组，虚线边框展示，自动计算日期范围
4. **便签管理**：单日图钉(pin)和日期范围(range)两种类型
5. **里程碑管理**：菱形图标标记关键节点
6. **工具栏导航**：年份切换，任务计数，新建入口，导入/导出
7. **右键菜单**：任务条右键编辑/删除/标记完成

## 3. 核心流程

### 3.1 智能行合并算法

```
输入：任务列表（可能时间重叠）
输出：每个任务分配的行号 + 总行数

1. 主线任务(isMain=true)优先排列在第0行
2. 其余任务按开始日期升序排序
3. 贪心遍历：尝试放入第一个不重叠的行
4. 若所有现有行都重叠，新增一行
5. 重叠判定：taskStart < existingEnd && taskEnd > existingStart
```

### 3.2 跨月任务切割

```
输入：带行号的年度任务列表
输出：12个月的 MonthLayout（含切割后的 TaskSegment/NoteSegment/MilestoneInMonth 列表）

1. 遍历每个任务，确定其在当前年份的起止月份
2. 按月份边界切割成片段
3. 标记 isStart/isEnd 用于圆角和箭头渲染
4. 同样处理便签和分组范围
5. 里程碑直接映射到对应月份的日期位置
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：莫兰迪色系（高明度、低饱和度），10组配色
- **强调色**：#5b6af0（蓝紫色主按钮），#EF4444（主线任务红色），#F59E0B（便签琥珀色），#FBBF24（里程碑金色）
- **按钮风格**：圆角6px，主按钮蓝底白字，次按钮灰底深灰字
- **字体**：-apple-system, BlinkMacGroupFont, 'Segoe UI', 'Inter', sans-serif
- **基础字号**：12px，标签9-11px，标题15px
- **布局风格**：顶部工具栏 + 下方滚动日历区，白卡片灰底
- **圆角**：卡片8px，按钮6px，任务条5px，颜色选择器圆形

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |

### 4.3 响应式

- 桌面端优先设计，最小宽度 800px
- 日历区域自适应宽度，任务条按百分比定位
- 平板端（≤768px）适当缩小字号
- 移动端（≤480px）工具栏换行，月份行可横向滚动

## 5. 数据持久化

- 使用 LocalStorage 存储所有数据
- 键名：`smart-timeline-data`
- 数据结构：`{ tasks: Task[], groups: TaskGroup[], notes: Note[], milestones: Milestone[] }`
- 每次增删改操作后 debounce 300ms 自动保存
- 页面加载时自动读取
- 支持导出为 .json 文件下载
- 支持导入已有 JSON 数据文件
